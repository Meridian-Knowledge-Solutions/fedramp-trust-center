# FedRAMP Trust Center

A React-based compliance monitoring dashboard for FedRAMP (Federal Risk and Authorization Management Program) systems. Provides real-time transparency into security controls, authorization status, and organizational policies for the Meridian Learning Management System.

## Tech Stack

- **React 18** with **Vite 7** — fast builds and HMR
- **Tailwind CSS 3** — utility-first styling
- **Recharts / Chart.js** — data visualization
- **Framer Motion** — animations
- **react-markdown** with remark-gfm — policy and report rendering
- **gh-pages** — GitHub Pages deployment

## Getting Started

```bash
npm install
npm run dev        # Start dev server
npm run build      # Production build
npm run preview    # Preview production build
npm run deploy     # Deploy to GitHub Pages
```

## Features

**Platform**
- Compliance dashboard with KSI validation grid and trend charts
- Trust center with live SCN (System Change Notice) tracking
- VDR security metrics dashboard
- Transparency console with real-time validation logs
- Pipeline metrics and failure history analytics
- Unified MAS (Moderate Assessment Scope) dashboard

**Organization**
- Policies viewer with full markdown rendering
- Schema browser for data validation rules
- Reports hub — OAR, QAR, VDR, and SCN reports in HTML and JSON formats
- Quarterly review scheduling with ICS calendar export

## Deployment

Deployed to GitHub Pages at `https://meridian-knowledge-solutions.github.io/fedramp-trust-center/`.

A CI workflow (`.github/workflows/codecommit.yml`) mirrors the repository to AWS CodeCommit on push to `master`. Dynamic data files in `public/data/` are generated by backend pipelines and excluded from version control.

## API

The frontend connects to an AWS API Gateway backend for authentication, package downloads, and system status. API configuration is in `src/config/api.js`.
